# CookingApp FrontEnd

React + Vite frontend scaffold for the CookingApp backend API.

## Requirements

- Node.js `20.19.x`

## Run locally

```bash
nvm use
npm install
npm run dev
```

## API routes currently used (no auth)

- `GET /api/recipes`
- `POST /api/recipes`

## Environment

Override backend URL with:

```bash
VITE_API_BASE_URL=http://localhost:3000
```
